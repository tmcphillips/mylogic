## 2019-07-20 Reset WSL Debian environment prior to setting up development environment

### Background
- The development environment for MyLogic needs to be easily reproducible.
- The first computing platform to support is Debian/Ubuntu which also runs in the Windows Subsystem for Linux environment.

### Recreated the WSL Debian environment from scratch
- On circe-win10 reset the Debian environment as follows.
- Right-clicked on `Debian GNU/Linux` in Start menu and selected `More -> App Settings`.
- Clicked `Reset` button, which causes it to "be reinstalled and revert to its default settings".
- Started Debian app from Start menu, entered Unix name and password at prompt.
